## 실제 브라우저 경계 검사

공개 Pages에서 `?motion=off`로 모든 의미 단계를 최종 상태로 만든 뒤 75장을 순회했다. 각 활성 슬라이드의 `scrollWidth`, `scrollHeight`와 가시 자식 요소의 경계 상자를 슬라이드 경계와 비교했다.

- 1366×768: 75장 중 위반 0장
- 1600×900: 75장 중 위반 0장
- 1920×1080: 75장 중 위반 0장
- 390×844: 문서·슬라이드 가로 overflow 0장

모바일 1차 검사에서는 세로 스크롤바 폭을 포함한 `100vw` 때문에 15px 가로 스크롤이 생겼고, 14번 방법론 행렬이 760px 최소 폭을 강제했다. 덱을 가용 폭 `100%`에 맞추고 행렬을 방법론별 요약 카드로 재배치한 뒤 전체를 다시 검사했다.

## 상호작용과 접근성

- `Space`·`→`: 현재 의미 단계를 하나씩 공개하고 마지막 단계 뒤 다음 슬라이드로 이동
- `←`: 현재 단계를 되돌리고 첫 단계 앞에서 이전 슬라이드로 이동
- `R`: 현재 슬라이드의 단계를 처음부터 다시 시작
- `O`·`Esc`: 오버뷰, `N`: 발표자 노트, `Home`·`End`: 처음·끝 이동
- `?motion=off`와 `prefers-reduced-motion`: 이동 없이 모든 의미 단계를 즉시 표시
- 인쇄·오버뷰: 의미 정보의 최종 상태 표시

텍스트와 의미 색은 가장 어두운 본문 표면을 기준으로 자동 계산했다. 작은 텍스트에 쓰이는 `ink`, `muted`, `dim`과 다섯 의미 색 모두 WCAG 대비 4.5:1 이상이다. 의미 SVG에는 `role="img"`와 한국어 요약을 제공하며, 색만으로 관계를 구분하지 않는다.
