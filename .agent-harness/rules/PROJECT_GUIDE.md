# seminar_ai_ontology 프로젝트 지침

## 프로젝트 개요

이 저장소는 기술 실무자가 온톨로지 기반 지식관리를 기초부터 실습까지 독학할 수 있는 한국어 공개 교안입니다. 결과물은 GitHub Pages용 정적 HTML 슬라이드와 배포용 PDF입니다.

## 작업 원칙

1. 콘텐츠 사실은 W3C 표준, 원 논문, 방법론 공식 문서 등 1차·공식 출처로 검증하고 `docs/references.md`에 기록합니다.
2. 반도체 regression failure 시나리오를 관통 예제로 사용하되 특정 회사의 비공개 지식이나 상표를 사실처럼 만들지 않습니다.
3. 한 화면에 하나의 교육 목표를 두고, 본문은 한국어로 작성하며 원어 전문용어를 병기합니다.
4. 애니메이션은 이해를 돕는 단계 공개에 사용하고 `prefers-reduced-motion` 및 키보드 탐색을 지원합니다.
5. 사용자의 기존 변경과 `main`의 선행 작업을 보존합니다.

## 검토 게이트

각 모듈은 별도 브랜치와 PR로 제출합니다. 사용자의 리뷰와 승인을 받기 전에는 다음 모듈 구현이나 병합으로 넘어가지 않습니다.

## 검증

```powershell
py -3 -m unittest discover -s tests -v
git diff --check
```

후속 모듈에서 빌드 명령이 추가되면 `.agent-harness/harness.config.json`도 함께 갱신합니다.
