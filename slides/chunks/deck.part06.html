SPARQL
25

SPARQL — graph에게 질문한다

최근 timeout 실패 조회
SELECT ?failure ?test ?evidence WHERE {
  ?failure a ex:TimeoutFailure ;
           ex:observedIn ?run ;
           ex:failedTest ?test ;
           ex:hasEvidence ?evidence .
  ?run ex:startedAt ?date .
}
ORDER BY DESC(?date)
LIMIT 20
